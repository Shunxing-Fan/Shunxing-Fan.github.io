# THAI1000 Lesson 2

Static Thai who-and-where practice site based on `Lesson 2 Text.pdf`.

The site plays generated Google Cloud Text-to-Speech MP3 assets first and falls back to the browser `speechSynthesis` API with `th-TH` when an asset is unavailable.

Generated voice: `th-TH-Chirp3-HD-Achernar`. The 44 vocabulary items and 10 conversation lines each have normal and slow MP3 variants.
