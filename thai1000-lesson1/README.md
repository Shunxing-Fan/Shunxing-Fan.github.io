# THAI1000 Lesson 1

Static Thai greetings practice site based on `Lesson 1 Text Part 2-3.md`.

The site plays generated Google Cloud Text-to-Speech MP3 assets first and falls back to the browser `speechSynthesis` API with `th-TH` when an asset is unavailable.

Generated voice: `th-TH-Chirp3-HD-Achernar`. Each vocabulary item and conversation line has normal and slow MP3 variants.
