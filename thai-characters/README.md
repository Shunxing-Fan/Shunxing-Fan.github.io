# Thai Characters

Static GitHub Pages site for THAI1000 Thai consonants, basic vowels, and compound vowels.

- Canonical URL: `https://shunxing-fan.github.io/thai-characters/`
- Data source: `assets/characters.json`
- Local audio: MP3 assets under `assets/audio/`, including Week 5-6 vowel pronunciations from FunToLearnThai.
- Offline behavior: service worker app shell cache plus user-triggered audio pack caching

The original generator remains in `scripts/generate_audio.py` for maintenance, but the current local audio pack includes downloaded MP3 vowel assets.
