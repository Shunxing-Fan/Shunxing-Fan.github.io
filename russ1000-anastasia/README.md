# RUSS1000 Anastasia Reading

Static reading page for the Anastasia Russian passage.

- `assets/content.json`: sentences, sentence tokens, word-level translations, and Chinese translation.
- `assets/audio/manifest.json` and `assets/audio/*.mp3`: offline audio pack.
- Generated voice: `ru-RU-Chirp3-HD-Charon`.
- Audio scope: sentence-level and word-level only.
- Variants: `normal` and `slow`.
