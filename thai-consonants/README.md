# Thai Consonant Sounds

Static study page for Thai consonant phoneme frequency.

- `assets/consonants.json`: 21 LT-CS phoneme rows and 42 active Thai consonant letters. Obsolete `喔僠 and `喔卄 are excluded.
- `assets/audio/manifest.json` and `assets/audio/*.mp3`: offline audio pack.
- Generated voice: `th-TH-Chirp3-HD-Achernar`.
- Each phoneme group and each letter name has normal and slow MP3 variants.
