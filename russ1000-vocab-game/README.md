# RUSS1000 Vocabulary Game

Static Russian vocabulary memorization game.

- `assets/vocabulary.json`: current Russian vocabulary list from the screenshots plus the added class words.
- `assets/audio/manifest.json` and `assets/audio/*.mp3`: offline audio pack.
- Generated voice: `ru-RU-Chirp3-HD-Charon`.
- Modes:
  - Listen to Russian audio and choose English.
  - See English and choose Russian; correct answers play audio.
